Это программа сохраняет занесенные данные в сортированном виде с помощью MySQL
Поддерживает команды:
$ manager add (game, book, film) "Название игры/книги/фильма" Пример ($ manager add game "Dota 2")
$ manager print(game, book, film) Выводит список всех игр/книг/фильмов. Пример ($ manager print game)
$ manager print -s (game, book, film) Выводит список всех игр/книг/фильмов в сортированом порядке от последнего добавления. Пример ($ manager print game)
$ manager status (game, book, film)  "Название игры/книги/фильма". Выводит статус выбранной игры/книги/фильма.
$ manager add status (game, book, film) "Название игры/книги/фильма". В следующем окне будет предложено изменить статус выбранной игры/книги/фильма
$ manager edit (game, book, film) "Название игры/книги/фильма". В следующем окне будет предложено изменить название выбранной игры/книги/фильма
$ manager remove (game, book, film) "Название игры/книги/фильма". Удалит выбранную игру/книгу/фильм.
